## Running the app

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

## Building the app

```bash
npm run build
```

The built files will be in the `dist/` directory.

## Data Storage

This app uses localStorage for data persistence. All your exercises, workout templates, and workout sessions are stored locally in your browser.
